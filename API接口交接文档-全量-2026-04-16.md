# API接口交接文档（全量）
更新时间：2026-04-16  
项目：Triangle（`backend`）

## 1. 基础信息
- 服务入口：`http://<host>:58085`
- 健康检查：`GET /health`
- API 文档页：`GET /api-docs`
- 路由前缀：绝大多数接口以 `/api` 开头
- 内容类型：`application/json`（上传接口除外）

## 2. 统一响应格式
所有接口统一返回：

```json
{
  "success": true,
  "code": 0,
  "message": "ok",
  "data": {},
  "timestamp": 1710000000000
}
```

错误时：
- `success=false`
- `code` 为业务错误码（见 `backend/src/utils/response.js`）
- HTTP 状态码由错误码映射（如 400/401/403/404/409/500）

## 3. 鉴权规则
- 支持两种 token 头：
  - `Authorization: Bearer <token>`
  - `X-Token: <token>`
- 鉴权中间件：
  - `authenticate`：必须登录
  - `optionalAuthenticate`：可匿名访问，带 token 则校验
  - `requireAdmin`：必须管理员

## 5. 分页与查询约定

- `pageSize`：每页条数，常见上限 100

常见分页返回：
- `list`
- `total`
- `page`
- `pageSize`
- `totalPages`

## 6. 近期关键变更（交接重点）
- 会员升级接口 `POST /api/payment/upgrade-membership`：
  - 后端已按等级 `sortOrder` 禁止降级和平级升级。
- 管理后台新增活跃 IP：
  - `GET /api/admin/active-ips`
  - 返回地区字段，地区由本地库 `geoip-lite` 解析。
- 管理后台站内信模板：
  - 模板支持 `usageCondition` 使用条件。
  - 群发接口支持按 `userStatuses`、`membershipLevels` 筛选。

## 7. 安全与运维注意事项
- `GET /api/feedback` 当前无鉴权，若不希望公开读取需补鉴权。
- `/api/assets/images/local-upload` 依赖内部上传密码，生产必须配置强密码。
- 所有上传接口有速率限制与文件校验，Nginx 同时需设置 `client_max_body_size`。
- token 有效期为 7 天（JWT）。

## 8. 与 Swagger 的关系
- `/api-docs` 可用，但文档可能滞后于代码。
- 交接联调请以 `backend/src/routes/*.js` 与控制器校验逻辑为准。
